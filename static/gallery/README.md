# Gallery photos

These six images appear in two places:

1. The **"Our Work" strip** on the homepage (horizontal scroll).
2. The full grid on the **/gallery** page.

Each file ships as a branded "Photo Coming Soon" tile, so the gallery always
looks intentional. **To publish a real photo, replace the file with the same
name and push** — no code changes needed.

| File | Recommended |
|------|-------------|
| `placeholder-1.jpg` … `placeholder-6.jpg` | JPG, ~1200×900, landscape (4:3). Keep the exact same file names. |

Photos are cached by browsers for up to an hour, so a swap can take up to an
hour to show for repeat visitors after it deploys.
