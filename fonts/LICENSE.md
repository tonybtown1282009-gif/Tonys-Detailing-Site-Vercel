# Font licenses

Both font families shipped here are licensed under the **SIL Open Font
License 1.1** (<https://openfontlicense.org>); license metadata is embedded
in each font file.

| Family | Weights | Source |
|--------|---------|--------|
| Inter | Regular 400, Medium 500 (Latin subset), SemiBold 600 | The Inter Project Authors — <https://github.com/rsms/inter> (400/500 via `@fontsource/inter`) |
| Montserrat | Regular 400 – ExtraBold 800 | The Montserrat Project Authors — <https://github.com/JulietaUla/Montserrat> |

The `.woff2` files are Latin-subset builds of the `.ttf` files, generated by
[`tools/build_fonts.py`](../tools/build_fonts.py).
